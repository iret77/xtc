Open the ClimbX dashboard.

Invoke the `climbx-dashboard` skill from the climbx-cowork plugin and follow it: publish the
single-file dashboard artifact with overview KPIs, recent posts, format and niche performance, the
account's voice and cadence, and the weekly posting schedule, cached for instant load and refreshed
in the background.

If no API key is configured yet, run `/climbx-setup` first.
